Tipos de Divs

O que são Divs?

Divs são como caixas onde guardamos algum conteúdo, quando nós precisamos manipular algum elemento dentro do HTML é preciso fazer isso de forma especifíca, se eu quero alterar um único texto eu preciso fazer isso de uma forma que não altere os outros também, é ai que entram as Divs. Elas são uma caixa que ao definir uma classe ou um ID eu consigo saber em qual 'caixa' está aquele elemento e consigo manipular ele separadamente. Como exemplo eu coloquei duas Divs com a mesma classe, enquanto a Div desse exemplo está com uma classe diferente, assim se eu quero manipular esse texto eu consigo fazer isso pelo nome da classe.

O que são Divs Semânticas

Já entendemos que Divs são caixas e que podemos dar nomes a estas caixas e escolher qual tipo de conteúdo colocar nelas. Acontece que o próprio HTML já tem algumas Divs com um propósito definido. Para este exemplo vou criar uma tag de lista para mostrar.

  • Div - Caixa padrão para armazenar qualquer tipo de elemento
  • Section - Mesma função que a Div padrão, porém é mais bem vista pelo sistema
  • Nav - Essa tag é usada para colocar o menu de navegação do site
  • Header - Toda a parte de cima do texto, o cabeçalho do site
  • Article - Div onde é armazenado textos de artigos
  • Aside - Um extensão do texto na tag Article
  • Span - Uma tag que funciona como div, mas no caso usada para armazenar pequenos trechos de texto
  • Footer - Rodapé da página
  • Form - Div que agrupa todo o conteúdo de um formulário
  • iFrame - Div que apresenta na tela algum conteúdo externo, permite assister um vídeo do YouTube sem sair da página, por exemplo
  • Input type="" - Um input é um campo que receberá algum tipo de dado, existem vários tipos de inputs para diferentes dados
